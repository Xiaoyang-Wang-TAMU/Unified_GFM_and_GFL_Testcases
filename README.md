# Unified_GFML_Testcase
This repository includes all the Simulink test case and Hardware-in-the Loop (HIL) test case for unified GFM/L model proposed in our paper. 
## Case1:individual inverter connected with an infinite bus; 
## Case 2: Two-inverter system (PQ-Vf) and (PV-Qf). 
## Case 3: IEEE39 bus system.
For Case 3, the system data is included in "IEEE39data_UGFandGFM.m".

# HIL Test
The HIL PLECS simulation file is also included in "HIL_UGFMLPredictAIO.plecs". It can be either run in local PC or in PLECS RT box.
Please refer to our paper for more information:

Xiaoyang Wang, Xin Chen. *A Unified Framework for Hybrid Grid-Forming and Grid-Following Inverter Control*. *arXiv preprint*, 2026. [arXiv:2607.01616.](https://arxiv.org/abs/2607.01616)
